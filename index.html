Mapping Mistakes

Circle Size

Designers usually can not size circles correctly.

When sizing the circle, you should double the area not the radius which will ends up quadruple the area.

Software Default

Designers usually rely on the software defaults to do visualizations.

But we should never trust Software Defaults. This is valid for any tools, not just for mapping programs.

Symbol placement

In proportinal symbol maps, some areas may get so crowded that it becomes difficult to figure out what's going on.

Make sure that small symbols are always in front of larger ones or make all symbols semi-transparent.
